# litemint.io - Japanese JP
Game URL：**[https://litemint.io](https://litemint.io)**

## DEX Tradable Cards
 - EP = エナジー | DP = ダメージ | HP = 回復する | AP = アーマー
 - **Bold name**: Legendary
 
## パッシブカードとアクティブカード
 - パッシブカード -　試合中に出すことはない、デッキに入れておくことで自動的に発動されるカード。(下記リストに"Passive"と表記)
 - アクティブカード - 試合中に自分のターンで出すことで効果を発揮するカード。
## Mastery、Droid、Spec

- それぞれ3種類ずつあるが、一つにデッキには各1種類しか入れることはできない。(Repair Masteryをデッキに入れたらOverload MasteryとHellfire Masteryは使えない)
